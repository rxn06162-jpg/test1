# 気象情報マップアプリケーション

OpenStreetMapとOpen-Meteoを利用し、クリックした地点の現在の天気と24時間予報を表示するシングルページアプリケーションです。

## 使い方
1. `index.html` をブラウザーで開きます。
2. 地図上の任意の地点をクリックします。
3. 左側に位置情報と現地時刻、下部に現在の天気、右側に24時間分のヒートマップ予報が表示されます。

## 主な機能
- OpenStreetMap/Leafletによるインタラクティブマップ
- Open-Meteo APIからの現在の天気と24時間予報取得
- Nominatim APIによる逆ジオコーディング
- 気温・降水量・風速のヒートマップ表示（色分け）

## 注意事項
- インターネット接続が必要です。
- APIの呼び出し頻度によってはレート制限の影響を受ける可能性があります。
